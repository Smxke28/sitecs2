#  Site CS2

Site informativo e de treino sobre Counter-Strike 2, desenvolvido com Next.js, TypeScript e Tailwind CSS.

##  Funcionalidades

- Conteúdo informativo sobre CS2
- Interface moderna e responsiva
- Construído com App Router do Next.js 16

##  Como executar

```bash
git clone https://github.com/Smxke28/sitecs2.git
cd sitecs2
npm install
npm run dev
```
Acesse `http://localhost:3000`
